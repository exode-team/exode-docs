## Параметры запроса

Эндпоинт возвращает **постраничный** список заказов (инвойсов) школы для аналитики по деньгам.

<Info>
  Один заказ = одна строка. Суммы (`totalAmount` / `discountAmount`) считаются по позициям этого селлера.
</Info>

### Фильтры

<ParamField query="types" type="array" required={false}>
  Фильтр по типам заказа. Массив значений enum типа инвойса (например `Regular`).
</ParamField>

<ParamField query="createdAtDateRange" type="object" required={false}>
  Период по дате создания заказа. Объект `{ from, to }`, даты в формате ISO.

  <Expandable title="Свойства createdAtDateRange">
    <ParamField query="from" type="string">
      Начало периода. Дата в формате ISO.
    </ParamField>

    <ParamField query="to" type="string">
      Конец периода. Дата в формате ISO.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField query="search" type="string" required={false}>
  Поиск по UUID/ID заказа, домену, email, телефону или имени покупателя.
</ParamField>

### Пагинация

<ParamField query="take" type="integer" required={false}>
  Количество записей на странице. Максимум `1000`, по умолчанию `100`.
</ParamField>

<ParamField query="page" type="integer" required={false}>
  Номер страницы, начиная с `1`.
</ParamField>

<ParamField query="skip" type="integer" required={false}>
  Количество пропускаемых записей. Альтернатива параметру `page`.
</ParamField>

        page: 1
      },
      headers: {
        'Seller-Id': '{{ sellerId }}',

    });

```php PHP
<?php

$url = 'https://api.exode.biz/saas/v2/invoice';
$params = [
  'types' => ['Regular'],
  'createdAtDateRange' => [
    'from' => '2026-05-01T00:00:00Z',
    'to' => '2026-05-31T23:59:59Z'
  ],
  'take' => 100,
  'page' => 1
];

$headers = [
  'Seller-Id: {{ sellerId }}',
  'School-Id: {{ schoolId }}',
  'Authorization: Bearer YOUR_TOKEN'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url . '?' . http_build_query($params));
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode === 200) {
  $result = json_decode($response, true);
  print_r($result['payload']);
} else {
  echo "Error: HTTP $httpCode\n";
  echo $response;
}
?>
```

```python Python
import requests
import json

url = 'https://api.exode.biz/saas/v2/invoice'

params = {
  'types': ['Regular'],
  'createdAtDateRange': { 'from': '2026-05-01T00:00:00Z', 'to': '2026-05-31T23:59:59Z' },
  'take': 100,
  'page': 1
}

headers = {
  'Seller-Id': '{{ sellerId }}',
  'School-Id': '{{ schoolId }}',
  'Authorization': 'Bearer YOUR_TOKEN'
}

try:
  response = requests.get(url, params=params, headers=headers)
  response.raise_for_status()

  result = response.json()
  print(json.dumps(result['payload'], indent=2, ensure_ascii=False))

except requests.exceptions.RequestException as e:
  print(f'Error: {e}')
  if hasattr(e, 'response') and e.response is not None:
    print(f'Response: {e.response.text}')
```

## Требования к правам доступа

<Check>
Для получения списка заказов требуется **право на продажи селлера** (`SellerSales`).
</Check>

<Warning>
  Сервисный пользователь должен быть аутентифицирован по токену и иметь соответствующие права доступа к указанной
  школе.
</Warning>

<Info>
  Возвращаются только заказы в рамках указанной школы. Суммы рассчитываются по позициям текущего селлера.
</Info>
