## Параметры запроса

Эндпоинт возвращает **постраничный** список курсов школы. Для каждого курса возвращается список id связанных групп (потоков).

<Info>
  У курса может быть несколько групп (потоков), поэтому `groupIds` — это массив.
</Info>

### Фильтры

<ParamField query="types" type="array" required={false}>
  Фильтр по типам курса. Массив значений enum типа курса (например `Course`).
</ParamField>

<ParamField query="search" type="string" required={false}>
  Поиск по названию курса.
</ParamField>

### Пагинация

<ParamField query="take" type="integer" required={false}>
  Количество записей на странице. Максимум `1000`, по умолчанию `100`.
</ParamField>

<ParamField query="page" type="integer" required={false}>
  Номер страницы, начиная с `1`.
</ParamField>

<ParamField query="skip" type="integer" required={false}>
  Количество пропускаемых записей. Альтернатива параметру `page`.
</ParamField>

```javascript Node.js
const axios = require('axios');

const listCourses = async () => {
  try {
    const response = await axios.get('https://api.exode.biz/saas/v2/course', {
      params: { types: ['Course'], search: 'Python', take: 100, page: 1 },
      headers: {
        'Seller-Id': '{{ sellerId }}',
        'School-Id': '{{ schoolId }}',
        'Authorization': 'Bearer YOUR_TOKEN'
      }
    });

    console.log('Courses:', response.data.payload);
  } catch (error) {
    console.error('Error:', error.response?.data || error.message);
  }
};

listCourses();
```

```php PHP
<?php

$url = 'https://api.exode.biz/saas/v2/course';
$params = [ 'types' => ['Course'], 'search' => 'Python', 'take' => 100, 'page' => 1 ];

$headers = [
  'Seller-Id: {{ sellerId }}',
  'School-Id: {{ schoolId }}',
  'Authorization: Bearer YOUR_TOKEN'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url . '?' . http_build_query($params));
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode === 200) {
  $result = json_decode($response, true);
  print_r($result['payload']);
} else {
  echo "Error: HTTP $httpCode\n";
  echo $response;
}
?>
```

```python Python
import requests
import json

url = 'https://api.exode.biz/saas/v2/course'

params = { 'types': ['Course'], 'search': 'Python', 'take': 100, 'page': 1 }

headers = {
  'Seller-Id': '{{ sellerId }}',
  'School-Id': '{{ schoolId }}',
  'Authorization': 'Bearer YOUR_TOKEN'
}

try:
  response = requests.get(url, params=params, headers=headers)
  response.raise_for_status()

  result = response.json()
  print(json.dumps(result['payload'], indent=2, ensure_ascii=False))

except requests.exceptions.RequestException as e:
  print(f'Error: {e}')
  if hasattr(e, 'response') and e.response is not None:
    print(f'Response: {e.response.text}')
```

## Описание полей

<ResponseField name="courseId" type="integer">
  ID курса.
</ResponseField>

<ResponseField name="name" type="string">
  Название курса.
</ResponseField>

<ResponseField name="type" type="string">
  Тип курса.
</ResponseField>

<ResponseField name="groupIds" type="integer[]">
  Массив id групп (потоков) курса. У одного курса может быть несколько групп.
</ResponseField>

## Требования к правам доступа

<Check>
Для получения списка курсов требуется **право на управление пользователями школы** (`SchoolManageUsers`) или **права куратора курса** (`CourseCurator`).
</Check>

<Warning>
  Сервисный пользователь должен быть аутентифицирован по токену и иметь соответствующие права доступа к указанной
  школе.
</Warning>

<Info>
  Возвращаются только курсы в рамках указанной школы.
</Info>
