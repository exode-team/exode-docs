```
GET /saas/v2/form/layout/list
```

Требуется аутентификация и право `FormManage`.

<Info>
    Все параметры фильтрации необязательны. Без фильтров метод вернёт все макеты форм школы с учётом пагинации.
    Структура макета — как в ответе [`form/layout/create`](/ru/exode-api/school/form-layout/create).
</Info>

<Info>
    Параметры-массивы передаются повторением параметра: `modes=Form&modes=Signup`.
</Info>

## Параметры запроса

### Пагинация

<ParamField query="take" type="integer" required={false}>
    Количество записей на странице. От `1` до `1000`. По умолчанию `100`.
</ParamField>

<ParamField query="page" type="integer" required={false}>
    Номер страницы (начиная с `1`). Альтернатива `skip`.
</ParamField>

<ParamField query="skip" type="integer" required={false}>
    Количество записей, которые нужно пропустить. По умолчанию `0`.
</ParamField>

### Фильтрация

<ParamField query="layoutIds" type="integer[]" required={false}>
    Фильтр по ID макетов.
</ParamField>

<ParamField query="layoutUuids" type="string[]" required={false}>
    Фильтр по UUID макетов.
</ParamField>

<ParamField query="slugs" type="string[]" required={false}>
    Фильтр по slug'ам макетов.
</ParamField>

<ParamField query="modes" type="enum[]" required={false}>
    Фильтр по режиму макета: `Form` (заполнения формы), `Signup` (поля пользователя при регистрации),
    `Custom`, `Welcome`, `Participant` (анкета участника продукта).
</ParamField>

<ParamField query="statuses" type="enum[]" required={false}>
    Фильтр по статусу макета: `Draft`, `Published`.
</ParamField>

<ParamField query="productIds" type="integer[]" required={false}>
    Фильтр по ID продуктов (для макетов, привязанных к продукту, например анкет участника).
</ParamField>

<ParamField query="search" type="string" required={false}>
    Поиск по названию макета. Максимум 50 символов.
</ParamField>

### Сортировка

<ParamField query="id" type="enum" required={false}>
    Сортировка по ID: `ASC` или `DESC`.
</ParamField>

<ParamField query="createdAt" type="enum" required={false}>
    Сортировка по дате создания: `ASC` или `DESC`.
</ParamField>

## Поля ответа

<ResponseField name="payload" type="object">
    Постраничный список макетов форм.

    <Expandable title="Свойства payload">
        <ResponseField name="items" type="object[]">
            Массив макетов.

            <Expandable title="Свойства элемента">
                <ResponseField name="id" type="integer">ID макета.</ResponseField>
                <ResponseField name="uuid" type="string">UUID макета.</ResponseField>
                <ResponseField name="slug" type="string">Slug макета.</ResponseField>
                <ResponseField name="name" type="string">Название макета.</ResponseField>
                <ResponseField name="internalName" type="string | null">Внутреннее название.</ResponseField>
                <ResponseField name="note" type="string | null">Заметка.</ResponseField>
                <ResponseField name="mode" type="enum">Режим: `Form`, `Signup`, `Custom`, `Welcome`, `Participant`.</ResponseField>
                <ResponseField name="status" type="enum | null">Статус: `Draft`, `Published`.</ResponseField>
                <ResponseField name="config" type="object">Конфигурация макета (поля, настройки).</ResponseField>
                <ResponseField name="sellerId" type="integer">ID продавца.</ResponseField>
                <ResponseField name="isEdited" type="boolean | null">Признак наличия правок.</ResponseField>
                <ResponseField name="createdAt" type="string">Дата создания (ISO 8601).</ResponseField>
                <ResponseField name="updatedAt" type="string | null">Дата обновления (ISO 8601).</ResponseField>
            </Expandable>
        </ResponseField>

        <ResponseField name="page" type="integer">Текущая страница.</ResponseField>
        <ResponseField name="count" type="integer">Общее количество записей.</ResponseField>
        <ResponseField name="pages" type="integer">Общее количество страниц.</ResponseField>
        <ResponseField name="isFirst" type="boolean">Признак первой страницы.</ResponseField>
        <ResponseField name="isLast" type="boolean">Признак последней страницы.</ResponseField>
        <ResponseField name="next" type="object">Параметры следующей страницы (`skip`, `take`, `page`).</ResponseField>
        <ResponseField name="prev" type="object">Параметры предыдущей страницы (`skip`, `take`, `page`).</ResponseField>
    </Expandable>
</ResponseField>
