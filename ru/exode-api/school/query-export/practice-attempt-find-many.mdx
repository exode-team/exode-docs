**type:** `QUERY_EXPORT_TYPE_COURSE_LESSON_PRACTICE_ATTEMPT_FIND_MANY`

Это тип отчёта для метода [`query-export/generate`](/ru/exode-api/school/query-export/generate): передайте его
в поле `type`, а описанные ниже поля — в `variables`.

Выгружает список попыток выполнения практических заданий с оценками, статусами и информацией о студенте.

### Колонки отчёта

- ID пользователя
- Фамилия Имя
- Способ входа
- Номер телефона
- ID курса
- Название курса
- Куратор/проверяющий
- Урок
- Статус
- Кол-во верных ответов
- Общее кол-во вопросов
- Набранное кол-во баллов
- Общее кол-во баллов
- Процент полученных баллов
- Номер попытки
- Всего попыток
- Дата отправки
- Дедлайн
- После дедлайна
- Дата создания
- Последняя активность

<Info>
    В экспорте формата **JSON** дополнительно отдаются технические поля: `email`, `attemptId`, `practiceId`,
    `studentSex`, `dateOfBirth`, `addedDate`, `lessonId`, `totalTasks`, `maxPointsAmount`, `pointsAmount`,
    `statusHistoryLogs`, `attemptUpdatedAt`.
</Info>
