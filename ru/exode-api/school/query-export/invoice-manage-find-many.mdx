**type:** `QUERY_EXPORT_TYPE_INVOICE_MANAGE_FIND_MANY`

Выгружает список счетов школы с суммами, типами, статусами оплаты и связанными платежами.

Это тип отчёта для метода [`query-export/generate`](/ru/exode-api/school/query-export/generate): передайте его
в поле `type`, а описанные ниже поля — в `variables`.

<Info>
  Набор фильтров совпадает с методом [Список счетов](/ru/exode-api/school/invoice/list). Итоговый контракт
  `variables` валидирует сервис экспорта — неизвестные поля игнорируются.
</Info>

---

### Переменные (`variables`)

<ParamField body="filter" type="object">
  Параметры фильтрации.

  <Expandable title="Свойства filter">
    <ParamField body="invoiceIds" type="integer[]">
      Фильтр по ID счетов.
    </ParamField>

    <ParamField body="userIds" type="integer[]">
      Фильтр по ID пользователей (плательщиков).
    </ParamField>

    <ParamField body="productIds" type="integer[]">
      Фильтр по ID продуктов.
    </ParamField>

    <ParamField body="types" type="string[]">
      Фильтр по типам счёта. Массив из значений: `Regular`, `InstallmentInit`, `InstallmentPay`,
      `SubscriptionInit`, `SubscriptionPay`.
    </ParamField>

    <ParamField body="search" type="string">
      Поиск по тексту, до 50 символов.
    </ParamField>

    <ParamField body="totalAmountRange" type="object">
      Диапазон суммы счёта. Свойства `from` и `to` — числа.
    </ParamField>

    <ParamField body="createdAtDateRange" type="object">
      Диапазон дат создания счёта. Свойства `from` и `to` — даты в формате ISO 8601.
    </ParamField>

    <ParamField body="payment" type="object">
      Вложенный фильтр по связанному платежу (статус, провайдер и т.д.).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="sort" type="object">
  Параметры сортировки. Значения: `ASC` (по возрастанию) или `DESC` (по убыванию).

  <Expandable title="Свойства sort">
    <ParamField body="id" type="string">Сортировка по ID. `ASC` | `DESC`.</ParamField>
    <ParamField body="createdAt" type="string">Сортировка по дате создания. `ASC` | `DESC`.</ParamField>
    <ParamField body="updatedAt" type="string">Сортировка по дате обновления. `ASC` | `DESC`.</ParamField>
    <ParamField body="status" type="string">Сортировка по статусу счёта. `ASC` | `DESC`.</ParamField>
    <ParamField body="totalAmount" type="string">Сортировка по сумме. `ASC` | `DESC`.</ParamField>
    <ParamField body="paidAtOrCreatedAt" type="string">Сортировка по дате оплаты (или создания). `ASC` | `DESC`.</ParamField>
    <ParamField body="userLastOnlineAt" type="string">Сортировка по последней активности плательщика. `ASC` | `DESC`.</ParamField>
    <ParamField body="paymentActualStatus" type="string">Сортировка по фактическому статусу платежа. `ASC` | `DESC`.</ParamField>
  </Expandable>
</ParamField>

  'variables' => [

  'Content-Type: application/json',
  'Authorization: Bearer YOUR_TOKEN'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode === 200 || $httpCode === 201) {
  $result = json_decode($response, true);
  echo "Execution UUID: " . $result['payload']['uuid'] . "\n";
} else {
  echo "Error: HTTP $httpCode\n";
  echo $response;
}
