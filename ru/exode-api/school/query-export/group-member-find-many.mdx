---

**type:** `GroupMemberFindMany`

Выгружает список участников групп с информацией о прогрессе по курсу, кураторе, тарифе и активности.

### Колонки отчета

- Студент
- Номер телефона
- ID пользователя
- Статус доступа
- Курс
- Прогресс %
- Пройдено уроков
- Всего уроков
- Куратор
- Текущий урок
- Тариф
- Дата создания доступа
- Последняя активность

---

```php PHP
<?php

$url = 'https://api.exode.biz/saas/v2/query-export/generate';
$data = [
  'type' => 'GroupMemberFindMany',
  'variables' => [
    'filter' => [
      'groupIds' => [10, 20],
      'active' => true
    ],
    'sort' => [
      'createdAt' => 'DESC'
    ]
  ]
];

$headers = [
  'Seller-Id: {{ sellerId }}',
  'School-Id: {{ schoolId }}',
  'Content-Type: application/json',
  'Authorization: Bearer YOUR_TOKEN'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode === 200 || $httpCode === 201) {
  $result = json_decode($response, true);
  echo "Execution UUID: " . $result['uuid'] . "\n";
} else {
  echo "Error: HTTP $httpCode\n";
  echo $response;
}
?>
```
