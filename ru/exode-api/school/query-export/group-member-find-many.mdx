title: "Участники группы"
description: "Выгрузка участников групп с прогрессом по курсу, кураторами и тарифами"
---

**type:** `QUERY_EXPORT_TYPE_GROUP_MEMBER_FIND_MANY`

Выгружает список участников групп с информацией о прогрессе по курсу, кураторе, тарифе и активности.

Это тип отчёта для метода [`query-export/generate`](/ru/exode-api/school/query-export/generate): передайте его
в поле `type`, а описанные ниже поля — в `variables`.

### Колонки отчета

- Студент
- Номер телефона
- ID пользователя
- Статус доступа
- Курс
- Прогресс %
- Пройдено уроков
- Всего уроков
- Куратор
- Текущий урок
- Тариф
- Дата создания доступа
- Последняя активность
