**type:** `QUERY_EXPORT_TYPE_PRODUCT_BILLING_ACCESS_FIND_MANY`

Выгружает биллинговые записи доступов к продуктам: статусы списаний (подписки/рассрочки), даты старта,
завершения и повторных попыток списания.

Это тип отчёта для метода [`query-export/generate`](/ru/exode-api/school/query-export/generate): передайте его
в поле `type`, а описанные ниже поля — в `variables`.

<Info>
  Фильтр связан с методом [Список доступов к продуктам](/ru/exode-api/school/product-access/list). Итоговый
  контракт `variables` валидирует сервис экспорта — неизвестные поля игнорируются.
</Info>

---

### Переменные (`variables`)

<ParamField body="filter" type="object">
  Параметры фильтрации.

  <Expandable title="Свойства filter">
    <ParamField body="billingIds" type="integer[]">
      Фильтр по ID биллинговых записей.
    </ParamField>

    <ParamField body="userIds" type="integer[]">
      Фильтр по ID пользователей.
    </ParamField>

    <ParamField body="invoiceIds" type="integer[]">
      Фильтр по ID счетов.
    </ParamField>

    <ParamField body="statuses" type="string[]">
      Фильтр по статусам биллинга. Массив из значений: `Scheduled`, `Processing`, `WaitingEntryFee`, `Paid`,
      `Failed`, `CanceledByUser`, `CanceledByFailedAutoCharge`, `CanceledByFailedManualCharge`,
      `CanceledByPreviousCancellation`, `CanceledByReinitialization`.
    </ParamField>

    <ParamField body="finishDateRange" type="object">
      Диапазон дат завершения доступа. Свойства `from` и `to` — даты в формате ISO 8601.
    </ParamField>

    <ParamField body="retryDateRange" type="object">
      Диапазон дат повторной попытки списания. Свойства `from` и `to` — даты в формате ISO 8601.
    </ParamField>

    <ParamField body="access" type="object">
      Вложенный фильтр по доступу к продукту (продукт, тариф, группа и т.д.).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="sort" type="object">
  Параметры сортировки. Значения: `ASC` (по возрастанию) или `DESC` (по убыванию).

  <Expandable title="Свойства sort">
    <ParamField body="id" type="string">Сортировка по ID. `ASC` | `DESC`.</ParamField>
    <ParamField body="createdAt" type="string">Сортировка по дате создания. `ASC` | `DESC`.</ParamField>
    <ParamField body="updatedAt" type="string">Сортировка по дате обновления. `ASC` | `DESC`.</ParamField>
    <ParamField body="status" type="string">Сортировка по статусу биллинга. `ASC` | `DESC`.</ParamField>
    <ParamField body="startDate" type="string">Сортировка по дате старта. `ASC` | `DESC`.</ParamField>
    <ParamField body="finishDate" type="string">Сортировка по дате завершения. `ASC` | `DESC`.</ParamField>
    <ParamField body="billingPercent" type="string">Сортировка по проценту оплаты. `ASC` | `DESC`.</ParamField>
  </Expandable>
</ParamField>
