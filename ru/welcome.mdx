Exode — EdTech SaaS-платформа для онлайн-школ и продюсерских центров. Exode API позволяет управлять
пользователями, группами, курсами, доступами, формами и выгрузками, а также получать события платформы через
вебхуки.

## Официальный SDK

<Card title="@exode-team/sdk" icon="npm" href="/ru/exode-sdk/overview">
    Официальный npm-пакет: типизированный серверный клиент для `/saas/v2/*` и bridge для встраиваемых
    мини-приложений (с React-хуками). Установка: `npm install @exode-team/sdk`.
    Доступен и на [npm](https://www.npmjs.com/package/@exode-team/sdk).
</Card>

## Возможности API

- 👥 Управление пользователями и группами школы
- 📚 Курсы, прогресс студентов и выгрузка попыток практик
- 🛒 Доступы к продуктам, счета и платежи
- 📝 Формы и кастомные поля пользователей
- 📊 Асинхронные отчёты и выгрузки (XLSX/CSV/JSON)
- 🔔 Вебхуки на ключевые события платформы
